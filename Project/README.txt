ReadMe:
1.运行环境 :  Ubuntu  64位.

2. 运行方式 : 关于makefile. 里边包含3个命令。
　　- make
　　- clean
　　- execute
	请先 make， 然后执行 make execute.
	如果有问题。请先 make clean 清空， 然后 make 编译文件， 最后执行 make execute.

3. 我们采用的是文件输出。本程序能够在相应的数据集文件夹下生成.output文件，无需重定向。其中dataset文件夹中存放所有aql文件以及对应的数据集文件夹。文件夹中包含有单个或多个.input文件。还包含有对应的.output文件，是我们的程序运行得到的结果。

4.我们实现了报错功能。具体有如下方面：
　　-  main函数提供找不到文件或文件夹的报错功能
　　-  lexer提供非法字符或非法标识符报错功能, 并定位错误的行和列。
　　-  parser 如aql中输出不存在的view或引用不存在的列。会进行报错。
　　
5.我们对代码进行了优化。优化方面具体如下：
　　- 定义特定的类PatternClass.h，辅助实现pattern的拼表功能，大大减少拼表时间。
　　- 使用特定的全局变量保存特定结果。防止重复调用，大大减少了实现各种功能的时间。
　　- 对于extract，先读取from语句，使得在读取前面其它语句时可以同步处理，减少存储空间。
